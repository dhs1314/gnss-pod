# 软件架构 — Sequential EKF Reduced-Dynamic POD v2.2.1

## 系统概述

序贯扩展卡尔曼滤波（EKF）精密单点定位（PPP）系统，逐历元处理GPS双频观测数据，估计LEO卫星轨道、经验加速度、对流层延迟、接收机钟差和浮点模糊度。

## 状态向量 (ECI坐标系)

| 分量 | 符号 | 维数 | 模型 | 过程噪声 |
|------|------|------|------|----------|
| 位置 | r | 3 | 轨道动力学 | 未建模加速度 1e-7 m/s² |
| 速度 | v | 3 | 轨道动力学 | 同上 |
| 经验RTN加速度 | aR,aT,aN | 3 | Gauss-Markov τ=600s | σ_ss=1e-8 m/s² |
| 天顶湿延迟 | zwd | 1 | 随机游走 | σ=1e-9 m/√s |
| 接收机钟差 | clk | 1 | 白噪声(每历元重置) | 全重置 |
| 浮点模糊度 | amb_1..N | N_sv | 常数 | 0 |

总维数 = 11 + N_sv

## 处理流程

```
每个历元:
  1. PREDICT: 轨道积分(dt) + STM传播 → x_pred, P_pred
  2. 钟差重置: clk=0, P_clk=1e10
  3. CODE UPDATE (每个SV, 标量):
     - 观测模型: P_if = ρ + c*clk + tropo
     - 无模糊度项
     - 锚定接收机钟差
  4. PHASE UPDATE (每个SV, 标量):
     - 观测模型: L_if = ρ + c*clk + tropo + amb
     - 钟差已被code约束
     - 模糊度可正确分离
  5. 新SV加入: 单历元on-the-fly bias估计
  6. 失联SV移除: >10历元未见则从状态中删除
```

## 动力学模型

- **地球引力**: GGM05C Nmax=90 (`src/gravity_model.py`)
- **经验加速度**: RTN方向 Gauss-Markov过程 (`src/empirical.py`)
  - 时间常数 τ=600s
  - 稳态标准差 σ_ss=1e-8 m/s²
- **数值积分**: DOPRI8 Runge-Kutta 7(8) (`src/orbit_integrator.py`)
  - 步长: 变步长, 容差 rtol=1e-12, atol=1e-14
  - 状态转移矩阵 (STM): 随轨道积分同时传播

## 观测模型

### 无电离层组合
- P_if = (f1²·P1 - f2²·P2) / (f1² - f2²)
- L_if = (f1²·L1 - f2²·L2) / (f1² - f2²)

### 观测方程
```
P_if = ρ + c·(clk_rcv - clk_sat) + ZHD·mf_h + ZWD·mf_w + B_sat
L_if = ρ + c·(clk_rcv - clk_sat) + ZHD·mf_h + ZWD·mf_w + amb
```

### 卫星码偏差 (sv_bias)
- 前60历元零均值中位数估计
- 新出现SV: 单历元时钟移除估计 `sv_bias = P_if + sat_clk - (ρ + sag + state.clk)`

## 对流层模型 (`src/troposphere.py`)

- **ZHD**: Saastamoinen (1972) 公式
  - LEO高度 (>80km): ZHD = 0
- **映射函数**: 简化GMF (Boehm et al., 2006)
  - 15°纬度带插值, 年周期余弦变化
  - Niell连分式形式

## EKF实现细节 (`src/sequential_filter.py`)

### 协方差传播
- Joseph-form: P = (I-KH)P(I-KH)ᵀ + K·R·Kᵀ
- 保证对称性, 防止数值发散

### 创新检验
- χ²检验: ν²/(HPHᵀ+R) < 10.828 (α=0.001, 1-dof)
- Code和Phase独立检验

### 关键配置参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| sigma_code | 0.3 m | 伪距观测噪声 |
| sigma_phase | 0.20 m | 载波相位观测噪声 |
| sigma_acc | 1e-3 m/s² | 未建模加速度(每历元) |
| Q_clk | 0.001*dt | 钟差过程噪声 (σ_rw=0.032 m/√s) |
| chi2_threshold | 100 | 创新χ²拒绝阈值 |
| interval | 30 s | 历元间隔 |

## 运行命令

```bash
py -3.12 run_sequential_pod.py --date 2024-04-29 --hours 0.5 --interval 30
```
