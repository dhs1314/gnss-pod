# GRACE-FO PPP 精密定轨系统 — 软件架构

**版本**: V2.2.0 (2026-05-31)
**概述**: 基于 GNSS 精密单点定位（PPP）的 GRACE-FO 卫星精密定轨系统，支持多种数据输入格式和处理策略。

---

## 一、数据输入层

系统支持 3 种 GPS 观测数据源：

| 数据源 | 格式 | 加载器 | 相位存储 | 伪距 | 码对齐 | 采样 | 整周模糊度 |
|--------|------|--------|---------|------|--------|------|-----------|
| **GPS1B ASCII** | `.pkl` 缓存 | `gps1b_loader.py` | 米 | 米 (C1 as P1) | 是 | 10s | 不可用 |
| **GPS1B RINEX** | `.rnx` 原始 | `gps1b_rnx_loader.py` | 整周数 | 米 (C1 as P1) | 是 | 10s | 不可用 |
| **GPS1A ASCII** | ISDC 下载 | `gps1a_loader.py` | 整周数 (1Hz) | 米 (0.1Hz) | 否 | 混合 | 理论可用 |

**关键差异**: JPL RL04 处理的 GPS1B 数据（ASCII 和 RINEX）将相位与 C/A 码对齐（`L1_phase ≈ P1`），整周模糊度信息已丢失，只能做浮点解。GPS1A 保留原始整周模糊度，但存在 ±13 km 每星伪距偏差，实际不可用。

### RINEX 2.20 格式细节 (GPS1B)

```
观测类型 (9种): L1 L2 C1 P1 P2 LA SA S1 S2
字段宽度: 16 字符/观测值，末位为 LLI (Loss of Lock Indicator)
每历元: 1 行 epoch header + 2 行/SV
L1/L2: 载波相位整周数 (~1.15×10⁸ cycles)
C1:    C/A 码伪距，米
P1:    P 码伪距，米
P2:    P 码伪距，米
```

### 外部辅助数据

| 数据类型 | 来源 | 用途 | 精度 |
|----------|------|------|------|
| **GNV1B** | ISDC/GFZ RL04 | 参考真值（ECEF 坐标，1s） | ~2 cm |
| **SP3 Final** | IGS/BKG | GPS 精密轨道+钟差（15min） | ~2.5 cm |
| **广播星历** | CDDIS | GPS 广播轨道（V1.0 遗产） | ~1 m |

## 二、PPP 处理管线

### 管线 1: `run_gps1b_ppp.py` — ASCII GPS1B + KF 浮点模糊度（主力）

```
GPS1B ASCII (.pkl)
        │
        ▼
┌──────────────────────────────────────────────────────┐
│ Pass 1: 偏差估计（前 60 个采样历元）                   │
│                                                      │
│   每星 DCB 偏差:  sv_bias[sv] = median(P_r)           │
│   浮点模糊度:     B_if[sv]    = median(L_if - P_if)    │
│                                                      │
│   其中: P_r = P_if + clk_sv - rho_corr               │
│         L_r = L_if + clk_sv - rho_corr               │
└──────────────────────────────────────────────────────┘
        │
        ▼
┌──────────────────────────────────────────────────────┐
│ Pass 2: Kalman 滤波（历元递推）                       │
│                                                      │
│ 状态向量:                                             │
│   [dX, dY, dZ, clk_r, trop_wet, B_G01..B_G32]        │
│    ├─ 临态 (4): dX, dY, dZ, clk_r — 每历元独立       │
│    └─ 持久 (1+N): trop_wet, B_Gxx — 跨历元继承       │
│                                                      │
│ 过程噪声 (30s 间隔):                                  │
│   Q_trop = 1e-6 m²/s  (~1 cm/√hr 随机游走)           │
│   Q_B    = 1e-4 m²/s  (跟踪 ~1 m/4h 漂移)            │
│                                                      │
│ 测量噪声:                                             │
│   σ_phase = 0.01 m  (相位: W=10000)                  │
│   σ_code  = 0.30 m  (伪距: W=11.1)                   │
│                                                      │
│ 测量模型:                                             │
│   Phase: L_if + clk_sv - rho_corr - sv_bias           │
│          = -e·dr + clk_r + trop·mf + B_sv             │
│   Code:  P_if + clk_sv - rho_corr - sv_bias           │
│          = -e·dr + clk_r + trop·mf                    │
│                                                      │
│ 测量有效性检查:                                       │
│   Phase: |L_r - sv_bias - B_if| < 200 m              │
│   Code:  |P_r - sv_bias| < 100 m                     │
│                                                      │
│ SV 生命周期:                                          │
│   新 SV: 缓冲 3 历元 → 初始化 B_if                    │
│        缓冲 5 历元 → 估计 sv_bias (在线)             │
│   失锁 >10 历元 → 从状态移除                          │
└──────────────────────────────────────────────────────┘
        │
        ▼
  输出: dE/dN/dU 误差 vs GNV1B, 3D RMS 统计
```

### 管线 2: `run_gps1b_rnx_ppp.py` — RINEX GPS1B + KF 浮点模糊度（V2.2.0 新增）

```
GPS1B RINEX (.rnx)
        │
        ▼
┌──────────────────────────────────────────────────────┐
│ RINEX 2.20 解析 (gps1b_rnx_loader.py):               │
│                                                      │
│   解析 epoch header (yy mm dd HH MM SS.sssssss)       │
│   解析 16 字符观测字段 (末位 LLI 提取)                │
│   L1/L2 整周数 × 波长 → 米                           │
│   IF 组合: L_if = α·L1 + β·L2                       │
│           P_if = α·P1 + β·P2                        │
│   ★ C1 优先于 P1 (匹配 JPL 码对齐)                   │
│   GPS 时标 → UTC (18s 闰秒)                          │
│   .pkl 缓存加速                                       │
└──────────────────────────────────────────────────────┘
        │
        ▼
  与管线 1 完全相同的 KF 算法
  精度等价（Δ < 0.01m 3D RMS，2024-04-29 验证）
```

### 管线 3: `run_gps1a_ppp.py` — GPS1A + 整周模糊度固定（实验性）

```
GPS1A ASCII (ISDC 下载, 1Hz 相位 + 0.1Hz 伪距)
        │
        ▼
┌──────────────────────────────────────────────────────┐
│ MW 宽巷模糊度:                                       │
│   N_w = L1_cyc - L2_cyc                              │
│         - (f1·P1 + f2·P2) / ((f1+f2)·λ_w)           │
│   精度: σ ~0.2 周                                     │
│                                                      │
│ N1 窄巷分解:                                         │
│   N1 = (B_if_mean - COEFF_W · N_w) / λ_n             │
│   ❌ 受 GPS1A 码偏差 (±13 km/SV) 影响，无法可靠固定   │
└──────────────────────────────────────────────────────┘
        │
        ▼
  KF with ambiguity-corrected phase
  ❌ GPS1A 伪距偏差漂移，整周固定不可行
```

### 管线 4: `run_ppp.py` (V1.0 遗产) — 广播星历 PPP

```
广播星历 (brdc*.24n RINEX nav) + 简化圆轨道
        │
        ▼
┌──────────────────────────────────────────────────────┐
│ PPPProcessor (src/ppp.py):                           │
│                                                      │
│   物理改正:                                           │
│     · 光行时 (2 次迭代 + 地球自转旋转矩阵)            │
│     · Sagnac 效应                                     │
│     · 相对论时钟 (Einstein + 二阶 Doppler - 重力红移) │
│     · 天线 PCV (高度角余弦模型)                       │
│     · 对流层 (Saastamoinen 干延迟 + 湿延迟状态估计)   │
│                                                      │
│   最小二乘: [X, Y, Z, clock, trop_wet]               │
│   高度角定权: w = 1/(σ²/(sin²(el) + 0.01))           │
└──────────────────────────────────────────────────────┘
        │
        ▼
  精度: ~11.8 cm 3D RMS (2024年5月全月仿真验证)
  ★ 广播星历为简化模型，非真实 GPS 观测
```

## 三、物理改正链

所有管线共享的物理改正流程：

```
伪距/相位原始值
    │
    ▼
┌─────────────────────────────────────────────────────┐
│ [1] 电离层自由组合 (IF Combination)                  │
│     L_if = α·L1 + β·L2                              │
│     消除一阶电离层延迟 (~99.9%)                      │
│     α = F1²/(F1² - F2²) ≈ 2.546                     │
│     β = -F2²/(F1² - F2²) ≈ -1.546                   │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│ [2] SP3 卫星位置插值                                 │
│     10 点 9 阶重心 Lagrange 插值                     │
│     15 分钟采样 → 任意时刻连续                       │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│ [3] 光行时改正                                       │
│     5 次迭代收缩至 1e-8 m 收敛                       │
│     rho_final = |sat(t - rho/C) - rcv(t)|           │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│ [4] Sagnac 效应                                      │
│     sag = (ωE/C) × (x_s·y_r - y_s·x_r)              │
│     地球自转导致信号传播路径弯曲                      │
│     ~30-50 m 量级                                    │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│ [5] 卫星钟差                                         │
│     从 SP3 内插获取                                  │
│     已含相对论周期项 (无需额外改正)                   │
│     ~0-300 km 量级                                   │
└─────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────┐
│ [6] 对流层湿延迟                                     │
│     映射函数: mf = 1/sin(el)                         │
│     状态估计 (随机游走)                              │
│     ~0-2 m 量级 (天顶方向)                           │
└─────────────────────────────────────────────────────┘
    │
    ▼
  观测量残差 = 几何距离 + 接收机钟差 + 对流层 + 模糊度
```

### 观测量定义

```
P_r = P_if + clk_sv - rho_corr    (伪距残差)
L_r = L_if + clk_sv - rho_corr    (相位残差)

其中:
  P_if = α·P1 + β·P2              (电离层自由伪距)
  L_if = α·L1 + β·L2              (电离层自由相位)
  clk_sv = SP3 内插卫星钟差 (米)
  rho_corr = |sat - rcv| + Sagnac  (几何距离 + Sagnac)

B_if = L_if - P_if ≈ 0            (码对齐后，浮点模糊度 ≈ 0)
```

## 四、坐标参考系

```
ECEF (Earth-Centered Earth-Fixed)
    → BLH (纬度/经度/高程，迭代 10 次至 1e-15 rad)
    → ENU (East-North-Up，局部切平面)

ENU 旋转矩阵:
  R = [ -sin(lon)          cos(lon)          0      ]
      [ -sin(lat)cos(lon)  -sin(lat)sin(lon)  cos(lat) ]
      [  cos(lat)cos(lon)   cos(lat)sin(lon)  sin(lat) ]

精度评估: dE, dN, dU (PPP - GNV1B) → 3D RMS
```

## 八、验证结果摘要 (2024-04-29 至 2024-05-01)

| 日期 | 管线 | 0.5h 3D RMS | 4h 3D RMS | <2m (4h) |
|------|------|-----------|-----------|---------|
| 04-29 | RINEX KF | 1.23m | 9.29m | 29% |
| 04-29 | ASCII KF | 1.22m | — | — |
| 04-30 | ASCII KF | 1.04m | 6.52m | 26% |
| 05-01 | ASCII KF | 2.99m | 9.80m | 18% |

### 已知限制

1. **浮点模糊度漂移**: B_if 在 4h 内漂移 0.2-1.7m，长弧段精度显著退化
2. **码对齐数据**: JPL RL04 相位 ≈ 码，无法整周模糊度固定，精度受码噪声 (~0.3m) 限制
3. **日变化**: 不同日期的可见 SV 集合不同，精度有日间差异 (1.0-3.0m)
4. **RINEX 覆盖**: 仅 2024-04-29 有 RINEX 文件
5. **D 星发散**: GRACE-FO D 卫星在弧段末端 KF 易发散

## 九、运行示例

```bash
# ASCII GPS1B 管线 (主力)
py -3.12 run_gps1b_ppp.py --date 2024-04-29 --hours 0.5 --interval 30 --grace-id C

# RINEX GPS1B 管线 (需有 .rnx 文件)
py -3.12 run_gps1b_rnx_ppp.py --date 2024-04-29 --hours 4 --interval 30 --grace-id C

# 批量处理
py -3.12 run_batch.py --ephem sp3 --start 2024-05-01 --end 2024-05-31

# GPS1A 整周模糊度 (实验性)
py -3.12 run_gps1a_ppp.py --date 2024-04-29 --hours 0.5 --interval 30
```
